# Python Full Course - Chapter 1

## Overview
This repository contains Python learning materials and code examples for **Chapter 1** of a comprehensive Python course.

## Topics Covered

### 1. **String Operations**
- String methods: `upper()`, `lower()`, `title()`
- String searching: `find()`, `count()`
- String manipulation: `replace()`, `endswith()`
- Escape sequences: `\n`, `\t`, `\\`, `\'`, `\"`

### 2. **Mini Projects**
- **Emoji Converter**: Convert text emoticons (`:)`, `:(`) into emoji characters (😊, 😒)

### 3. **Conditional Statements**
- `if`, `elif`, `else` statements
- Voting eligibility checker
- Grade calculator based on marks
- Number classification (Positive, Zero, Negative)

### 4. **Data Structures**
- **Lists**: Creation, indexing, slicing, built-in methods (`len()`, `max()`, `min()`)
- **Tuples**: Immutable sequences, indexing, methods like `index()`
- **Dictionaries**: Key-value pairs (file truncated)
